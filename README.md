# Productivity App

A comprehensive, all-in-one productivity application that seamlessly integrates a Personal Knowledge Management (PKM) system, time-blocking calendar, live task timer, academic reference management, and local AI capabilities.

## Features

- **Advanced PKM**: Robust Markdown engine with bidirectional linking and Graph View.
- **Time-Blocking**: Drag-and-drop scheduling and Google Calendar sync.
- **Task Management**: Task lists, priorities, and deadlines.
- **Local AI**: Privacy-first AI for summarization and action item extraction.
- **Zotero Integration**: Seamless academic reference management.

## Tech Stack

- **Platforms**: iOS 17+, macOS 14+
- **Database**: SwiftData
- **Sync**: CloudKit
- **Architecture**: MVVM / TCA
